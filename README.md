游戏安装方法  
1.下载apk文件安装  
2.手机新建Wa2Res文件夹大小写需要一致  
3.将pc版的文件夹移动到手机根目录重命名成Wa2Res即可  
4.0.2测试版开始不需要转换后的视频,之前的版本如果需要播放视频自行使用releases里的视频转换工具,将mv开头的pak文件转换成ogv视频,然后把转换好的视频放到Wa2Res里的movie文件夹里,或者下载我提供的ps3版视频  
5.游戏里还有部分动画特效未完成,不影响主线推进后续更新    
6.如果出现闪退问题清除缓存即可,0.1.8之前的版本不要清除用户数据,清除数据会清空存档,0.1.8版本开始存档储存在外部清除数据卸载存档也不会消失  
8.ps3版视频下载地址  pan.baidu.com/s/1UTSgchekEH0PVtM7yck94w 密码 qjqa
